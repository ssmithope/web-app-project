About This Project

Learn more about the goals and technologies used in this Web Apps module project.

Project Purpose

This project was created as part of the CSE 310 - Applied Programming course to fulfill the Web Apps module requirements. The goal is to demonstrate fundamental web development skills using HTML, CSS, and JavaScript.

Technologies Used

  • HTML5: For page structure and semantic layout.
  • CSS3: For styling, layout, colors, and responsive design.
  • JavaScript: For client-side interactivity and form validation.

Module Requirements Addressed

  • Multiple pages: Home, About, and Contact.
  • Semantic HTML elements for structure.
  • Custom CSS styling and responsive design.
  • Interactive JavaScript features (form validation and theme toggle).
